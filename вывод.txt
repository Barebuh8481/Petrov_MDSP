ключевые отличия:

1.объявление и наследование
python для наследования использует скобки, с# использует двоеточие

2.структура
python использует отступы(индексация), шарп использует фигурные скобки и точку с запятой, код вкладывается в пространства имен

3.типизация
питон динамически переопредлеяет всё что нужно в конструкторе, не нужно указывать типы данных
шарп статический типизированный, нужно объявлять типы данных внутри полей класса, до их использовния

4.конструкторы
в питоне всегда через __init__ принимающий первым аргументом self в виде ссылки на текущзий объект
шарп: называется как класс, имеет аналог self -- this, он необязателен для явного обозначения

5. полиморфизм
питон сам понимает метод по имени
в шарпе нужно опрделить метод виртуальным, в наследнике поментить override
